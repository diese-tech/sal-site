# SAL Product Brief

## Core Concept

Serpent Ascension League (SAL) is a persistent competitive Smite 2 Conquest league platform centered around live snake drafts.

The platform combines:

- league infrastructure
- public live draft presentation
- social player identity
- stream-friendly UX
- modular esports broadcast components

SAL is intentionally NOT:

- fantasy football software
- a spreadsheet admin panel
- a stat-heavy scouting system
- a trading card simulator

## Product Identity

"What if Discord had a premium esports draft platform?"

The product should feel:

- social
- alive
- competitive
- modular
- cinematic
- readable on stream

## UI Philosophy

### Identity First

Players should feel like people, not stat rows.

Primary emphasis:

- Discord avatar
- banners
- role identity
- org affiliation
- personality tags
- draft history

### Broadcast First

The draft board should feel like a live esports event.

The board itself is part of the product identity.

### Layered UX

The roster board is always present.

Captain tools overlay the experience instead of replacing it.

### Modular Design

All major UI should be component-driven.

Examples:

- player cards
- org cards
- ghost queue cards
- roster slots
- recent pick widgets
- stream overlays

## Draft Philosophy

The draft is a live event.

Public spectators should:

- understand current draft state
- follow roster evolution
- see recent picks
- identify active orgs

But should NOT:

- see private captain queues
- see future draft order
- see captain notes

## Long-Term Vision

Persistent player identities should support:

- season history
- championships
- org history
- captain history
- player profiles
- free agency

Without turning SAL into a social media platform.
